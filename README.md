gitstrap
========

A basic structure to start a JS project

    # install dependencies
    make dependencies

----------------------------------------

    # generic JS variable
    make var
    
    # node.js
    make node
    
    # AMD module
    make amd
    
    # all together plus tests via node.js
    make

----------------------------------------

    # test only
    make test
    
    # test via web (click on the generated url)
    make web
    
    # update pages test
    make pages

Latter one requires a `gh-pages` branch
